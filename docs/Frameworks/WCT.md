# Windows Community Toolkit (WCT)

## Summary

A collection of controls, helpers and extension for WinUI 2, WinUI 3, and Uno Platform developers. Simplifies and demonstrates common developer tasks building experiences for Windows with .NET.

**Website:** [WCT Docs](https://learn.microsoft.com/dotnet/communitytoolkit/windows)

### Languages

**Framework Languages:** C#

**App Languages:** C# and other .NET languages

### OS Support

**.NET:** Windows 10, and 11 as per .NET version

### Dependencies

The Windows Community Toolkit as a whole depends on:

- [.NET](https://dotnet.microsoft.com/download/dotnet)
- [XAML Behaviors](https://github.com/microsoft/XamlBehaviors) [Some packages: Behaviors]
- [Win2D](https://github.com/microsoft/Win2D) [Some packages: Media, ImageCropper]

### Canonical Apps

**.NET:**

- [Windows Community Toolkit Gallery](https://apps.microsoft.com/detail/9nblggh4tlcq)
